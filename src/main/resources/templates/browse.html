Questions
In the questions section you'll be able to browse user created exam questions.
Sections
Exam Questions can be arranged into sections. Candidates can be required to answer part of the questions in each section.
Papers
Sections are arranged into exam papers.
Attempts
A single attempt is a single candidate sitting a particular test paper on a given date in a given venue.
Exams
Exams are made up from multiple attempts for each student enrolled on a given module.
